# Project results

The complete hold-out comparison is stored in:

`results/model_comparison.csv`

Random Forest produced the strongest overall results across the five target
variables. The executed notebook contains the full printed outputs for
dataset exploration, VIF analysis, Lasso feature selection, regression-model
comparison and the manual `mels_S` polynomial experiment.

The results can be regenerated by running the scripts in `code/modelling/` from
the repository root.
